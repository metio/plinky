Marcar una partitura como aprendida a partir de la nota

Una partitura cuenta como aprendida cuando una ronda alcanza esta nota, lo que la programa para repaso. Baja el listón para marcar partituras como aprendidas antes.

Qué pasa con tus notas cuando te tomas un descanso.

Cuando dejas de practicar

Tus niveles nunca bajan: las piezas solo pierden su brillo hasta que las repasas. La opción amable por defecto.

Piezas para repasar de una vez

Cuántas piezas pendientes ofrece una sesión de repaso de una vez, para que siga siendo un lote acabable y no un montón.

Posición de dedos

Pequeños ayudantes que muestran qué tecla y qué dedo usar.

Mostrar la siguiente nota a tocar

Una tecla equivocada siempre parpadea en rojo. Esto define cuándo se enciende la tecla correcta: dejarlo oculto deja la lectura en tus manos.

Nombres de las notas en las teclas

Nombres de notas impresos en las teclas mientras aprendes dónde están. «Solo Do» marca solo las teclas de Do, la tecla blanca a la izquierda de cada par de teclas negras, como referencia para orientarte.

Lectura

Cómo se lee la música mientras tocas y las ayudas para encontrar la siguiente tecla.

Nivel

Tu propia mezcla de ayudas. Elige un nivel para restablecerlas.

Cada cabeza de nota toma el color de su nombre: todos los Do en rojo, todos los Re en naranja, y así sucesivamente, para que puedas leer por color mientras los nombres se te quedan.

Las notas que vienen se apilan sobre las teclas y caen sobre ellas mientras tocas, así ves cuál es la siguiente tecla.

La pieza se lee como una sola línea que se desplaza mientras tocas, en vez de filas apiladas.

Numera el primer compás de cada línea, para encontrar un pasaje fácilmente. Desactívalo para un pentagrama más limpio.

Compases/fila

Fuerza este número de compases por línea para una notación más grande; Automático los ajusta a tu pantalla.

Tamaño de las notas

Amplía toda la partitura para notas más grandes y fáciles de leer: útil en una pantalla pequeña, o cuando estás empezando.

Barras

Las barras unen notas rápidas en grupos de tiempo. Ocúltalas para leer una nota a la vez mientras aprendes; Automático las muestra en piezas más difíciles, donde el agrupamiento ayuda.

Una repetición translúcida de tu mejor intento en esta pieza se reproduce junto a ti.

Toca de oído: las notas empiezan ocultas y se revelan a medida que las encuentras, en verde cuando aciertas, en rojo cuando se acaban los intentos. Escucha primero y luego toca lo que oíste.

Metrónomo

El clic constante que te ayuda a llevar el ritmo.

Subdividir el pulso

Clics por pulso: 1 en el pulso, 2 para corcheas, 3 para tresillos, 4 para semicorcheas.
